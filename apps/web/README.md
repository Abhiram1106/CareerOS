# Frontend

This folder contains the Next.js web app. The repo uses `pnpm` workspaces, so
run install and dev commands from the repository root.

## Install

```bash
cd C:/Users/ADMIN/Desktop/Projects/CareerOS
pnpm install
```

## Run

```bash
cd C:/Users/ADMIN/Desktop/Projects/CareerOS
pnpm dev
```

Web app: [http://localhost:3000](http://localhost:3000)

## UI structure

- **Overview** — product hero (what CareerOS Campus AI is, demo loop, CTAs).
- **Workspace** — student console with tabs: Account, Resume & ATS, Readiness.
