# templates

Fill-in templates for each memory record type. Copy and fill in.

- `session-digest.md` — end-of-session record
- `error-entry.md` — bug fix → prevention rule
- `decision-entry.md` — architectural/product decision
- `project-context.md` — living project snapshot
- `daily-summary.md` — daily review
- `agent-note.md` — AI tool/model observation
